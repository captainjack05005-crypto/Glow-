# Beauty & Glow Studio

A modern, responsive luxury beauty salon website for **Beauty & Glow Studio** — a professional beauty services salon based in Salem, Tamil Nadu, India.

## Features

- Full-screen hero with parallax scroll effect
- Animated page-load screen
- Sticky navigation bar with mobile hamburger menu
- Services grid with image hover effects
- Testimonial slider with swipe support
- Masonry image gallery with lightbox
- Appointment booking form with live validation
- Scroll-reveal animations (Intersection Observer)
- Running marquee banner
- Contact section with social links
- Scroll-to-top button
- Fully responsive (mobile, tablet, desktop)
- SEO-optimised semantic HTML

## Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, CSS Grid, CSS animations
- **Vanilla JavaScript** — no dependencies
- **Google Fonts** — Cormorant Garamond (serif) + Jost (sans-serif)

## Running Locally

```bash
# Open directly in a browser
open index.html

# Or serve with any static server, e.g. npx serve
npx serve .
```

Or deploy instantly to Netlify — the `netlify.toml` sets the publish directory to `.` (repo root).
